# 燕云十六声 · 双生江湖

一款基于《燕云十六声》世界观的文字游戏合集，包含两种核心玩法。

## 🎮 游戏列表

### 📜 江湖行 · 剧情冒险
多线叙事，通过选择影响 **侠义值** 与 **武学悟性**，走向不同结局。
- 十几种结局
- 深度角色扮演
- 还原五代十国武侠氛围

### ⚔️ 百战燕云 · Roguelike地牢
永久死亡机制，随机武学，挑战幽冥渊十八层。
- 每局随机两门武学
- 奇遇事件
- 层数递增难度

## 🚀 在线试玩
项目部署到 GitHub Pages 后可直接访问。

## 📁 本地运行
直接双击 `index.html` 即可开始游戏（无需服务器）。
